# ToDoList
Simple todo list with PERN stack [WORK IN PROGRESS]
- For now, it has the simple functionality of creating, editing, completing, and deleting tasks.
  
## Stack
Back-End
- Node.js
- Express
- PostgreSQL

Front-End
- React with Javascript
- CSS
- Material-UI

## Dependencies
- Node
- PG
- Express
